# asu-progeng-frontend

Фронтенд-часть проекта онлайн-магазина

**Используемые технологии:**

- Vue.js - javascript фреймворк для динамического отображения UI
- vue-router + unplugin-vue-router - роутинг страниц с автоматической типизацией
- tanstack vue query - единая шина кэширования запросов
- VueUse - библиотека с полезными вспомогательными функциями
- valibot - создание схем валидации форм
- reka-ui + sonner - основа для создания UI компонентов
- tailwindcss + tailwind-merge + class-variance-authority + clsx - задание вариантов компонентов через атомарные классы и задание приоритета атомарных классов

## Запуск

1.  `pnpm install`
2.  `pnpm dev`

## Билд

1.  `pnpm install`
2.  `pnpm build`

## Структура проекта
Используется классическая структура из документации Vue. Все переиспользуемые компоненты лежат в папке `components`, все не переиспользуемые лежат в папке соответствующей страницы внутри папки `pages`
